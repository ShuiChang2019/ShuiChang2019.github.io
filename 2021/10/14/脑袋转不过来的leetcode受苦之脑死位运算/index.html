神秘题目1

2的幂

给你一个整数 n，请你判断该整数是否是 2 的幂次方。如果是，返回 true ；否则，返回 false 。

如果存在一个整数 x 使得 n == 2x ，则认为 n 是 2 的幂次方。

题解提到的技巧：

技巧1：n & (n - 1)

一个数 n 是 2 的幂，当且仅当 n 是正整数，并且 n的二进制表示中仅包含 1个 1。即，一个数n是2的幂，其二进制表示就是100000…（n个0）

提取1的方法 可以是**n & (n - 1)**。以1000（16）举例。n=1000, n-1=0111, n&(n-1)=0。

当有高位的1出现时，就会变成这样，以101000（28）为例。n=101000, n-1=100111, n&(n-1)=100000。

所以当n非2的幂次方时，n & (n - 1) 非0。

技巧2：n & (-n)

-n为n的补码。以1000（16）举例。n=00001000, -n=11111000, n & (-n)=n。

当有高位的1出现时，就会变成这样，以101000（28）为例。n=00101000, -n=11011000, n&(-n)=1000。

颠倒二进制位

颠倒给定的 32 位无符号整数的二进制位。

神秘·遍历从低到高查看给定的32位无符号整数的位数然后依此确定返回值从高到低的位数

神秘题解：位运算分治

若现有一二进制串 A = abcdefgh （其中abcdefgh为1或0）将其翻转可以做如下操作：

均分为左右两部分，对每部分递归执行翻转操作，然后将左半部分拼在右半部分的后面，即完成了翻转。就如题解所说的最底层递归的两部分：

1.取出所有奇数位和偶数位；

2.将奇数位移到偶数位上，偶数位移到奇数位上。

具体操作如下：

如：取

mask1 = 01010101

mask2 = 00110011

mask3 = 00001111

则 另一部分

mask1’ = 10101010

mask2’ = 11001100

mask3’ = 11110000

(mask1’ = mask1<<1; mask2’ = mask2<<2; mask3’ = mask3<<3)

每一步反转操作可分为三部分，第一步如下：

1.“过滤”：

A & mask1 = 0b0d0f0h;

A & mask1’ = a0c0e0g0;

2.位移：

0b0d0f0h << 1 = b0d0f0h0;

a0c0e0g0 >> 1 = 0a0c0e0g;

3.组合：

B = b0d0f0h0 || 0a0c0e0g = badcfehg;

对于倒数第二层，就每两位分一组。所做的操作如下：

B & mask2 = 00dc00hg;

B & mask2’ = ba00fe00;

00dc00hg << 2 = dc00hg00;

ba00fe00 >> 2 = 00ba00fe;

C = dc00hg00 || 00ba00fe = dcbahgfe;

对于倒数第三层，就每三位分一组。所做的操作如下：

C & mask3 = 0000hgfe;

C & mask3’ = dcba0000;

0000hgfe << 4 = hgfe0000;

dcba0000 >> 4 = 0000dcba;

D = hgfe0000 || 0000dcba = hgfedcba;

这就反转了。

异或·只出现一次的数字

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。
